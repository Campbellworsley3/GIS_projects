Campbell Worsley | GIS Projects

This Portfolio provides a few examples of interactive webmaps that I have created to inform users on varying issues
  • COVID-19 Weekly Cases By County

    This is a choropleth map created using QGIS, Python, JavaScript leaflet, and HTML. It displays the running case count for each county in the US for each week from 2020-2023.

  • Chapel Hill Neighborhood Median Housing Price

    This project utilized Python, QGIS, JavaScript, and HTML to create a visualization of the changing housing prices in Chaple Hill, NC from 2020-2025.

  • Manhattan Rent Pricing Map

    This project takes Manhattan leasing data and displays the change in leasing price before, during, and after the COVID-19 pandemic in an effort to inform companies of leasing trends.
